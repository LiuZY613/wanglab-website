## 💡 开发计划：小组成员技术分享

我们正在开发一个**课题组共享笔记本**功能。上线后，每位成员可以使用 GitLab 账号登录，直接在网页中撰写和发布内容。

**可以实现的事情：**
- 分享新学到的编程技巧或算法思路
- 推荐好用的科研工具、网站、效率方法
- 记录踩过的坑和解决方案
- 随时随地写 Markdown，一键发布，网站即刻更新

**实现方式**：基于 Decap CMS 的管理面板，组员浏览器登录 → 网页编辑器写 Markdown → 自动提交到仓库 → 自动构建并上线，全程无需离开浏览器。
